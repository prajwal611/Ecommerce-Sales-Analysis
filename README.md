🛒 E-commerce Sales Analysis Dashboard (Excel)
📖 Overview

This project is an Excel-based data analytics dashboard designed to analyze and visualize e-commerce sales data. It helps in tracking key business metrics such as revenue, profit, orders, and customer trends in a simple and interactive way.

🎯 Objectives
Analyze overall sales performance
Identify top-performing products and categories
Track monthly and yearly trends
Understand customer purchasing behavior
Enable quick decision-making using visual insights
🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Cleaning & Transformation
📈 Key Features
Interactive dashboard with slicers (Region, Category, Date)
KPIs: Total Sales, Profit, Quantity, Orders
Monthly and yearly sales trends
Top products and categories analysis
Region-wise performance insights
📂 Dataset
The dataset includes order details such as:
Order Date
Product Category
Sales Amount
Profit
Quantity
Region
🚀 How to Use
Download the Excel file
Open in Microsoft Excel (recommended 2016 or later)
Use slicers to filter data dynamically
Explore charts and KPIs for insights
📌 Insights Gained
Identified peak sales periods
Found high-performing product categories
Analyzed regional sales contribution
Detected trends for business growth
🔮 Future Improvements
